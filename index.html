Projects

Object Detection & Clustering - Fall 2022:
For my Web Information Search and Management class, we were assigned a semester long, group project where we had to identify, cluster, and extract texts given sets of ERD Diagrams. First, we identified the objects in the diagram by their classes. Then, we extracted texts from those objects. Finally, we ran multiple clustering algorithms based on their texts. I was able to detect the objects by their classes using the Tensorflow library and implemented Kmeans++ given the text files outputs.
This was my first big project using Python, different libraries, and Jupyter Notebook. I learned more about deep learning than I've ever had and became very comfortable with Python. With the freedom I was given, I was also able to select what methods or algorithms to implement to improve the overall accuracy and the runtime. .

Chess Game - Fall 2021:
Three other students and I created a chess game that mimicked chess.com. The game could save users' informations, their records, and their progress in any chess game that they have played. We also gave them points depending on their records. I was in charge of creating and maintaining the database/tables using PostgreSQL and pgAdmin. Additionally, I worked on the frontend and was able to display proper information using those tables.
This was my first time building an entire program from scratch and I learned a lot from other group members about servers, database incoporation, and even learned python. You can check out my code repository here .

Shell Interpreter - Spring 2021:
In a Systems Programming class, I was able to create a program that mimicked common shells including bash and csh with wildcarding. As a sophomore in college, I created multiple processes and handled file redirection, piping, and background processes. For languages, I used C for program development, C++ for string manipulation, and Lex & Yacc for the scanner and the parser.
This was the hardest class project that I have encountered in my first two years at Purdue but it was a good introductory project to learn about multiple processes, Unix programming, and piping. You can check out my code repository here .
